Dashboard Overview

The human body is source of thousands of types of bacteria and different parts of the body harbour different species. For example, bacteria found in a person's gut is not same as bacteria found on person's eyelashes. Bacterial species could even vary from the same location for different individuals.

One such study was done on bacterial species collected from navel area from 153 people. To keep the identity private, each of these individuals were provided an ID number. The following dashboards are created for study participants and researchers to see which bacterial species reside in participant's navel.

From the dropdown below, select an ID number for a study participant to view various information on the participant and their navel bacterial information.

Demographic Info

The bar chart displays the top 10 bacterial species (OTUs) for the selected participant ID.
Hover over each bar to know the names of the bacterial species displayed as OTUs.
The Gauge chart displays the weekly washing frequency's value. The black indicator bar display the value as a measure from 0-10 on the progress bar in the gauge chart when an participant ID is selected.

The bubble chart displays all the bacterial species collected for the selected participant ID. The bubble sizes are indicative of the sample size for each bacteria specie.
